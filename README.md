# Task Manager API

## Description

This is a simple Task Manager API built with Node.js and Express.js.

## Installation

1. Clone the repository:
   git clone <your-repo-url>
2. Install dependencies:
   npm install

## Usage

Start the server:
npm run dev

## Testing

Run tests:
npm run test

## Features

- Get all tasks
- Get task by id
- Create task
- Update task
- Delete task
